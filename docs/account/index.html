Y.NetLaboアカウントについて

「Y.NetLaboアカウント」は、Y.NetLabo のアプリを横断して使える共通アカウントです。ひとつのアカウントで、対応アプリのデータを引き継ぎ・同期できます。このページでは、アカウントの作成・管理・削除の方法をまとめています。

Y.NetLaboアカウントとは

Y.NetLaboアカウントは、下記の対応アプリで共通してご利用いただける無料のアカウントです。ID・パスワードを登録することで、機種変更やアプリの再インストール後もデータを引き継げます。

  • アプリを横断して同じアカウントでログインできます。
  • ログインせず「ゲスト」として使う場合、データはお使いの端末内にのみ保存されます。
  • LINE アカウントと連携してログインすることもできます（wa/ri・soroe の iOS版では、Apple でのサインインにも対応しています）。

マイページの使い方

アカウントの作成・ログイン、プロフィールの変更、外部アカウント連携、ログアウトなどは、すべて各アプリ内の「マイページ」から行えます。マイページは、画面右上（上部）のアカウントアイコンをタップすると開けます。

アカウントの作成・ログイン

ログインせずに「ゲスト」として使い始めることもできますが、ゲストの場合はデータがお使いの端末内にのみ保存されます。マイページからアカウントを作成、または既存のアカウントでログインすると、機種変更やアプリの再インストール後もデータを引き継げます。

  • ID・パスワードは、それぞれ5文字以上で設定してください。
  • 作成済みのアカウントは、ID・パスワードを入力していつでもログインできます。

アカウント連携（LINE・Apple）

マイページの「アカウント連携」から、LINE アカウントを連携できます。連携すると、他の端末でも同じアカウントでログインできます。

あわせて、wa/ri・soroe の iOS版では、Apple でのサインイン（Apple ID でのログイン）にも対応しています。ご利用の端末・アプリによって、利用できる連携方法は異なります。

データの引き継ぎ

ゲストとして利用していたときの記録は、アカウントの新規作成時、または既存アカウントへのログイン時に引き継ぐことができます。

外部アカウントの連携のみでは、ゲスト時の記録は引き継がれません。データを引き継ぐ場合は、アカウントの作成・ログインの操作を行ってください。

プロフィール（表示名）の変更

マイページから表示名を編集できます。表示名は、ルームの作成者名などとして表示される名前です。

お問い合わせID

マイページには「お問い合わせID」が表示されます。タップするとコピーできます。サポートへお問い合わせいただく際に、このIDをお知らせいただくとスムーズです。

ログアウト

マイページからログアウトできます。ログアウトするとゲストの状態に戻りますが、アカウントのデータは保持されており、再度ログインすれば元の状態に戻せます。

アカウントの削除

Y.NetLaboアカウントは、いつでも削除できます。削除すると、ユーザーアカウント情報および登録されたすべてのデータが即時に削除され、元に戻すことはできません。

方法1：アプリから削除する

  1. アプリを開き、画面上部のアカウントアイコンからマイページを開きます。
  2. 「削除して退会」ボタンをタップします。
  3. 確認のうえ実行すると、アカウント情報と登録されたすべてのデータがただちに削除されます。

方法2：お問い合わせから削除を依頼する

アプリから操作できない場合は、以下のいずれかの方法で削除をご依頼いただけます。担当者が確認次第、アカウント削除の手続きを行います（ご連絡までおおむね1週間程度いただく場合があります）。

  • お問い合わせフォームから削除を依頼する（返信用のメールアドレスをご記入ください）
  • メールで依頼する：ynetlabo@gmail.com

削除されるデータ

アカウント削除時には、次の情報が削除されます。

  • アカウント情報（ID・登録情報・外部アカウント連携情報）
  • アプリ上でアカウントに登録・作成したすべてのデータ

削除は取り消せません。必要なデータがある場合は、削除前にご確認ください。

データの取り扱い・プライバシー

アカウントに関連する情報の取り扱いについては、プライバシーポリシーに記載しています。ご不明な点は下記までお問い合わせください。

お問い合わせ

アカウントに関するご質問は、お問い合わせフォームまたは ynetlabo@gmail.com までご連絡ください。
